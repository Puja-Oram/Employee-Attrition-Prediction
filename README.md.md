\# **Employee Attrition Prediction**



\## Project Overview



This project predicts whether an employee is likely to leave the organization using Machine Learning.



\## Features



\- Data Preprocessing

\- Exploratory Data Analysis

\- Model Training

\- Employee Attrition Prediction



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Scikit-Learn

\- Matplotlib

\- Seaborn



\## Dataset



Employee HR Dataset containing employee information such as:



\- Age

\- Job Role

\- Monthly Income

\- Overtime

\- Job Satisfaction

\- Work-Life Balance



\## Model Used



Random Forest Classifier



\## How to Run



1\. Install dependencies



```bash

pip install -r requirements.txt

```



2\. Run model



```bash

python model.py

```
